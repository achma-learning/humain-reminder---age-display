# Mortality

A Chrome extension that shows a live counter of your age to motivate you to live life to the fullest.

## Build

1. `yarn install --frozen-lockfile`
1. `yarn build`
1. `yarn package`

You will find the built package under `web-ext-artifacts` folder.
